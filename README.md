Got a state machine?
Need to replay scenarios from said state machine?

## Usage

```js
var preservative = require('preservative');

```

## Installation
Install locally from npm

```bash
$ npm install preservative --save
```

## Running tests
Install development dependencies

```bash
$ npm install
```

Run the tests

```bash
$ npm test
```
